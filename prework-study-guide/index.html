HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • HTML is like the frame of the house. It gives structure to the webpage.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • The following steps is to save my work to GitHub:
    • navigate to prework-study-guide directory and use:
      • git add -A
    • Add a commit message by replacing "your message here"
      • git commit -m "your message here"
    • After it's been committed locally, make sure local branch is up to date before pushing up the changes. Pull in main branch to the feature branch:
      • git pull origin main
    • Complete by using git push to put up your changes to the remote repository:
      • git push origin feature/add-js
    • Go on GitHub. Check the changes we pushed! Pull them requests! Take your issues and CLOSE IT!
JavaScript
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Adds logic to the webpage. If we input something, we can create an output to reflect what we put in.
